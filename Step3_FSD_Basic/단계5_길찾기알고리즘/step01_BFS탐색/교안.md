## 1. 도입 (10분)

### 미로 탈출을 어떻게 할까?

전략 1: **무조건 오른쪽 벽을 따라가기** → 어떤 미로는 막힘!  
전략 2: **BFS** → 항상 최단 경로 보장!

### BFS = 물이 퍼지는 것처럼 탐색

```
    S = 출발점, G = 목표, # = 벽

    시작:          1단계:         2단계:
    # # # # #     # # # # #     # # # # #
    # S . . #     # S 1 . #     # S 1 2 #
    # # # . #     # # # . #     # # # 2 #
    # . . G #     # . . G #     # . . G #
    # # # # #     # # # # #     # # # # #
```

**BFS 특징**: 
- 가까운 곳부터 탐색 (레이어별로 확장)
- 처음 목표에 도달하면 → **그게 최단 경로!**

## 2. BFS 코드 이해 (20분)

### 기본 구조

```python
from collections import deque

def bfs(grid, start, goal):
    queue   = deque([start])    # 탐색할 위치 목록
    visited = {start: None}     # 방문한 위치 + 이전 위치 기록

    while queue:
        current = queue.popleft()  # 가장 앞에서 꺼내기

        if current == goal:
            return reconstruct_path(visited, start, goal)

        # 상하좌우 이웃 탐색
        for next_pos in get_neighbors(grid, current):
            if next_pos not in visited:
                visited[next_pos] = current  # 이전 위치 기록
                queue.append(next_pos)

    return []  # 경로 없음
```

### 왜 deque를 쓸까?
- `list.pop(0)` : O(n) — 느림
- `deque.popleft()` : O(1) — **빠름!**
