```
          y (+)
          ↑
          │
          │     ● 로봇 위치 (0.3, 0.5)
          │
──────────┼──────────→ x (+)
(0,0) 출발│
          │
          │
         (-)
```

- **(0, 0)** = 로봇 출발 지점
- **x** = 동서 방향 (오른쪽이 +)
- **y** = 남북 방향 (위쪽이 +)
- 단위: **미터(m)**

## 3. 피타고라스 정리 복습 (10분)

두 점 A(1, 0)과 B(4, 4) 사이의 거리는?

```
     B(4,4)
      │
    4 │  (빗변 = 거리)
      │
  A───────
  (1,0)  3
```

거리 = √(3² + 4²) = √(9+16) = √25 = **5**

파이썬으로:
```python
import math
dx = 4 - 1  # 3
dy = 4 - 0  # 4
distance = math.sqrt(dx**2 + dy**2)  # 5.0
```
