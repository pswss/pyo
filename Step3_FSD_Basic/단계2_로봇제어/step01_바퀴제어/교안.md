# Step 01 — 바퀴 제어

## 학습 목표
- 로봇이 어떻게 움직이는지 원리를 이해한다
- `move_wheels()` 함수로 직접 로봇을 조종한다
- 왼쪽/오른쪽 바퀴 속도 차이로 방향을 바꾸는 원리를 안다

---

## 1. 도입 (10분)

### 질문으로 시작하기
> "여러분이 자동차를 운전한다면 어떻게 방향을 바꾸나요?"

차는 앞바퀴를 꺾어서 방향을 바꿉니다.  
하지만 이 로봇은 **탱크처럼** 좌우 바퀴 속도 차이로 방향을 바꿉니다!

### 탱크형 구동 원리 (그림으로 설명)

```
왼쪽 빠름, 오른쪽 느림 → 오른쪽으로 회전
┌─────────────────────────────┐
│  ←← 빠름 ←←               │
│  로봇                       │
│  ←← 느림 ←←               │
└─────────────────────────────┘

왼쪽 = 오른쪽 속도 → 직진
┌─────────────────────────────┐
│  ←← 같음 ←←               │
│  로봇                       │
│  ←← 같음 ←←               │
└─────────────────────────────┘
```

## 2. 코드 분석 (15분)

### 실제 프로젝트 파일 열기
[src/robot/drive_base.py](../../../src/robot/drive_base.py) — 29번 줄

```python
def move_wheels(self, left_ratio, right_ratio):
    self.left_wheel.move(left_ratio)
    self.right_wheel.move(right_ratio)
```

### 속도값의 의미
- 범위: **-1.0 ~ +1.0** (비율, 최대속도의 몇 %)
- `+1.0` = 최대 속도 전진
- `-1.0` = 최대 속도 후진
- `0`    = 정지

### 실제 최대 속도
[src/robot/robot.py](../../../src/robot/robot.py) — 74번 줄
```python
max_wheel_speed = 6.28  # 라디안/초 (1회전 = 2π ≈ 6.28)
```

## 3. 표로 정리 (칠판에 함께 작성)

| left_ratio | right_ratio | 결과 |
|-----------|-------------|------|
| 1.0       | 1.0         | 직진 |
| -1.0      | -1.0        | 후진 |
| 1.0       | -1.0        | 제자리 왼쪽 회전 |
| -1.0      | 1.0         | 제자리 오른쪽 회전 |
| 1.0       | 0.5         | 오른쪽으로 커브 |
| 0.0       | 0.0         | 정지 |

---

## 핵심 정리

```
로봇 이동 = 왼쪽 바퀴 속도 + 오른쪽 바퀴 속도의 조합
같으면 직진, 다르면 회전, 반대면 제자리 회전
```
