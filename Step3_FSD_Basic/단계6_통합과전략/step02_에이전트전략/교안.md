# Step 02 — 에이전트 전략 & 대회 준비

## 학습 목표
- 3가지 서브에이전트 전략을 이해한다
- 우선순위 기반 전략 선택 원리를 안다
- 대회에서 점수를 높이기 위한 파라미터 튜닝을 해본다

## 1. 3가지 탐색 전략 (15분)

[src/agent/agent.py](../../../src/agent/agent.py)

### 전략 1: GoToFixtures — 피해자 먼저!
```python
# 감지된 피해자가 있으면 즉시 이동
# 점수가 높은 피해자를 우선
if detected_fixtures:
    go_to_nearest_fixture()
```

### 전략 2: FollowWalls — 벽 따라 탐색
```python
# 벽 근처를 따라가며 피해자 감지 범위 최대화
# 피해자는 주로 벽 근처에 있음!
follow_wall_margin()
```

### 전략 3: GoToNonDiscovered — 미발견 구역 탐색
```python
# BFS로 가장 가까운 미탐색 구역 찾기
# 지도를 최대한 완성!
target = find_nearest_undiscovered_area()
```

### 우선순위 조합
```python
# SubagentPriorityCombiner
priority = [
    GoToFixturesAgent,      # 우선순위 1: 피해자 이동
    FollowWallsAgent,       # 우선순위 2: 벽 탐색
    GoToNonDiscoveredAgent, # 우선순위 3: 미발견 탐색
]

# 1번이 실행 불가능하면 2번, 2번도 안 되면 3번
```

## 3. 대회 채점 방식 이해 (10분)

| 항목 | 점수 |
|------|------|
| 피해자(H) 발견 | 25점 |
| 피해자(S) 발견 | 10점 |
| 피해자(U) 발견 | 5점 |

- 남은 시간이 적으면? → 바로 복귀!
