## 1. 대회 규칙: 피해자 유형 (10분)

### RoboCup Rescue Maze 피해자

| 표시 | 이름 | 색상 | 점수 |
|------|------|------|------|
| H | Harmed (부상자) | 빨강+흰색 | 높음 |
| S | Stable (안정) | 노랑+초록 | 중간 |
| U | Unharmed (무사) | 초록+흰색 | 낮음 |

| 기호 | 의미 | 색상 조합 |
|------|------|-----------|
| 가연성 | 불이 붙는 물질 | 빨강 배경 |
| 독성 | 독성 물질 | 흰 배경 + 특정 패턴 |
| 부식성 | 부식성 물질 | 검정 배경 |

### 핵심 로직 — 색상 비율로 피해자 판별

```python
# 이미지에서 각 색상의 픽셀 수를 센다
white_pixels  = count_color(img, "white")
black_pixels  = count_color(img, "black")
red_pixels    = count_color(img, "red")
yellow_pixels = count_color(img, "yellow")

total = white + black + red + yellow

# 색상 비율로 피해자 타입 결정
if red_pixels / total > 0.3:      # 빨강이 30% 이상
    return "H"  # Harmed
elif yellow_pixels / total > 0.3: # 노랑이 30% 이상
    return "S"  # Stable
elif white_pixels / total > 0.5:  # 흰색이 50% 이상
    return "U"  # Unharmed
```

### 실제 탐지 흐름

```
카메라 이미지
      ↓
BGR → HSV 변환
      ↓
각 색상 마스크 생성 (빨강, 노랑, 초록, 흰색, 검정)
      ↓
픽셀 수 계산
      ↓
비율 계산 → 피해자 타입 분류
      ↓
Executor에 보고 → 대회 심판 시스템에 전송
```
