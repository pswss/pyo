## 2. RGB vs HSV (15분)

### RGB — 빛의 혼합

| R (빨강) | G (초록) | B (파랑) | 결과 색상 |
|---------|---------|---------|---------|
| 255 | 0 | 0 | 빨강 |
| 0 | 255 | 0 | 초록 |
| 255 | 255 | 0 | 노랑 |
| 255 | 255 | 255 | 흰색 |
| 0 | 0 | 0 | 검정 |

**문제**: 조명이 바뀌면 같은 빨간 물체도 RGB 값이 달라짐!

### HSV — 인간의 색 인식 방식

- **H** (Hue): 색조 (빨강=0, 노랑=30, 초록=60, 파랑=120...)
- **S** (Saturation): 채도 (0=회색, 255=선명한 색)
- **V** (Value): 밝기 (0=검정, 255=밝음)

**장점**: 조명이 달라져도 H(색조)는 거의 변하지 않음!

```python
import cv2 as cv
import numpy as np

img_bgr = cv.imread("image.jpg")
img_hsv = cv.cvtColor(img_bgr, cv.COLOR_BGR2HSV)  # HSV 변환

# 빨간색 범위 (HSV 기준)
lower_red = np.array([0,   100, 100])
upper_red = np.array([10,  255, 255])

mask = cv.inRange(img_hsv, lower_red, upper_red)  # 필터링
```
